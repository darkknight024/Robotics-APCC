FLAGS SUMMARY
-------------
--mode p          : Playback mode (required)
--csv <file>      : CSV file path (required)

DEFAULT BEHAVIOR
----------------

TYPICAL WORKFLOW
----------------

4. Review speed analysis in speed_data/

5. Temp files cleaned up automatically
